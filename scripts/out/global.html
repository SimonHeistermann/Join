doTheyMatch(password, confirmation)

Überprüft, ob die eingegebenen Passwörter übereinstimmen. Diese Funktion vergleicht die Werte der Passworteingabefelder. Wenn die Passwörter übereinstimmen, wird die Funktion `submitUser()` aufgerufen, um den Benutzer zu registrieren. Falls die Passwörter nicht übereinstimmen, werden beide Felder geleert, und eine Fehlermeldung wird dem Benutzer angezeigt.
Parameters:
Name Type Description
password HTMLInputElement Das Eingabefeld für das Passwort des Benutzers.
confirmation HTMLInputElement Das Eingabefeld für die Passwortbestätigung des Benutzers.

enableRegisterButton(checkbox, button, isChecked)

Aktiviert den "Registrieren"-Button, wenn die Checkbox zur Zustimmung der Datenschutzrichtlinie angehakt ist. Diese Funktion überprüft, ob die Checkbox mit der ID `agree_privacy_policy` angehakt ist. Ist dies der Fall, wird der "Registrieren"-Button mit der ID `register_button` aktiviert, indem das `disabled`-Attribut auf `false` gesetzt wird.
Parameters:
Name Type Description
checkbox HTMLInputElement Die Checkbox, die anzeigt, ob die Datenschutzrichtlinie akzeptiert wurde. Sie wird über die ID `agree_privacy_policy` ausgewählt.
button HTMLButtonElement Der Button, der aktiviert wird, wenn die Checkbox angehakt ist. Er wird über die ID `register_button` ausgewählt.
isChecked boolean Ein Boolean-Wert, der den aktuellen Zustand der Checkbox (angekreuzt oder nicht) angibt.

handleUsers(storedUserArray, users)

Lädt das Array der gespeicherten Benutzer aus dem `localStorage` und gibt es in der Konsole aus. Diese Funktion holt sich die Benutzerdaten aus dem `localStorage`, die unter dem Schlüssel `users` gespeichert sind. Die Daten werden als JSON-String abgerufen und dann mit `JSON.parse` in ein JavaScript-Array umgewandelt. Falls keine Benutzerdaten im `localStorage` gefunden werden, wird `storedUserArray` den Wert `null` haben. Das geparste Array der Benutzer wird anschließend in der Konsole ausgegeben.
Parameters:
Name Type Description
storedUserArray string Der JSON-String, der die Benutzerinformationen aus dem `localStorage` enthält.
users Array | null Das JavaScript-Array, das die geparsten Benutzerdaten enthält. Falls keine Daten vorhanden sind, ist es `null`.

registerUser(users, form, formData, userObject, userID)

Registriert einen neuen Benutzer, indem die Formulardaten gesammelt und ein Benutzerobjekt mit einer eindeutigen ID erstellt wird. Der Benutzer wird anschließend in das `users`-Array eingefügt.

formData FormData Ein FormData-Objekt, das die Formulardaten als Schlüssel-Wert-Paare speichert.

userID number Die eindeutige ID für den neuen Benutzer, basierend auf der aktuellen Anzahl der Benutzer im `users`-Array.
